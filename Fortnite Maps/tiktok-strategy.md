## Clip Editing Tips

### Software
- **CapCut** (free, TikTok-integrated, perfect for this)
- **DaVinci Resolve** for anything more complex

### Style Guide
- **Fast cuts** ‚Äî no clip longer than 3 seconds unless it's a build-up to a scare
- **Subtitles on everything** ‚Äî 80% of TikTok is watched muted. Use CapCut auto-captions, then style them (bold, colored keywords)
- **Sound design matters more than music** ‚Äî for horror clips, use ambient dread + sudden loud sounds. For tycoon/building clips, use trending audio
- **The 1-second rule:** If the first second doesn't hook, they're gone. Start mid-action or with a bold text overlay

### Formats That Work
- **Before/After:** Empty lot ‚Üí finished park (satisfying)
- **POV:** First-person gameplay with text narration
- **Storytime + Gameplay:** Talk over clips ("So I built this horror map...")
- **Reaction Duets:** React to your own scares or others playing
- **Tutorial:** Quick tips, always start with "Here's something most people don't know..."

### Technical
- Record at 1080p minimum, 60fps preferred
- Use Fortnite's replay mode for cinematic shots
- Vertical crop (9:16) ‚Äî frame the action in center
- Brightness UP for horror clips (TikTok compression darkens everything)
